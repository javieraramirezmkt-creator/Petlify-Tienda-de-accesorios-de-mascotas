# 🐾 PetShop Chile - E-commerce Vue

Landing page tipo e-commerce para una tienda de mascotas en Chile, desarrollada con Vue 3 + Vite.  
Incluye catálogo de productos, carrito funcional y diseño moderno responsive.

---

## 🚀 Tecnologías utilizadas

- ⚡ Vue 3 (Composition API)
- ⚡ Vite
- 🎨 CSS moderno (variables + responsive)
- 🛒 Estado global simple (carrito)

---

## 🧩 Funcionalidades

- 🐶 Catálogo de productos con imágenes
- 🛒 Carrito funcional:
  - Agregar productos
  - Contador dinámico
  - Lista de productos en carrito
- 📱 Diseño responsive (mobile + desktop)
- 🎨 UI moderna estilo e-commerce
- 🔗 Navegación con scroll suave
